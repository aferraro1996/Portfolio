Random Password Generator & Sorter

This project allows a user to input 2 variables, X & Y, to determine the length and amount of passwords to generate, respectively.
It prints out all passwords generated, as well as printing them in Radix Sort.

Python Script:


import random
import string

def generate_passwords(length, amount):
    pool = string.ascii_letters + string.digits
    passwords = [''.join(random.choice(pool) for _ in range(length)) for _ in range(amount)]
    return passwords

def radix_sort(passwords, length):
    pool = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz"
    char_to_index = {char: index for index, char in enumerate(pool)}
    for position in range(length - 1, -1, -1):
        buckets = [[] for _ in range(62)]
        for password in passwords:
            index = char_to_index[password[position]]
            buckets[index].append(password)
        passwords = [password for bucket in buckets for password in bucket]
    return passwords
